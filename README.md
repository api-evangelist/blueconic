# BlueConic

BlueConic is a customer data platform (CDP) that liberates first-party data from
disparate systems and makes it available wherever it is needed to transform customer
relationships. Over 300 companies worldwide, including Hearst Newspapers, Heineken,
ING, T-Mobile, and VF Corp, use BlueConic to unify customer data into persistent,
individual profiles and activate that data across touchpoints.

## REST API

BlueConic provides a REST API v2 that allows developers to programmatically access
and manage:

- **Visitor Profiles** - Unified customer profile data
- **Segments** - Dynamic audience groupings
- **Interactions** - Customer engagement events
- **Audit Events** - Platform activity logging

The API uses OAuth 2.0 for authorization, supports both client credentials and
authorization code flows, and is fully OpenAPI-compliant. JSON is the only supported
request/response format.
